================================================================================
  KEY CHANGES REFERENCE
================================================================================

SERVERS (CRITICAL - Breaking Change)
   V2.6: servers: [{url, protocol}, ...]    (Array)
   V3.x: servers: {prod: {host, port}, ...} (Map)
   Impact: Change from .getElements() to .propertyMap().values()

OPERATIONS (STRUCTURAL - Major Change)
   V2.6: channels[ch].publish/subscribe
   V3.x: operations[op].{action, channel, message}
   Impact: Different navigation paths, action field validation

CHANNELS (ADDITIVE)
   V3.x: New 'address' field (required)
   Impact: Add validation rule for V3 channels

ACTION (NEW V3)
   V3.x: operation.action = "send" | "receive"
   Impact: Create new validation rule

TAGS/DOCS (Stable)
   Same structure, different locations (in operations in V3)
   Impact: Reuse validation logic, update navigation paths
