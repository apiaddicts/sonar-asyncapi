The description in 'info' should start with a capital letter.

Noncompliant Code Example (AsyncAPI)

asyncapi: '2.3.0'
info:
  title: Swagger Petstore Events
  version: '1.0.0'
  description: esta es una descripción que no comienza con mayúscula 
servers:
  production:
    url: broker.mycompany.com
    protocol: mqtt
channels: {}

Compliant Solution (AsyncAPI)

asyncapi: '2.3.0'
info:
  title: Swagger Petstore Events
  version: '1.0.0'
  description: Esta es una descripción que comienza con mayúscula 
  production:
    url: broker.mycompany.com
    protocol: mqtt
channels: {}
