Summaries must start with a capital letter.

Noncompliant Code Example (AsyncAPI)

  asyncapi: '2.3.0'
  info:
    version: '1.0.0'
    title: Swagger Petstore Events
  servers:
    production:
      url: broker.mycompany.com
      protocol: mqtt
  channels:
    petAdded:
      description: "an event for when a pet is added" # El resumen debe comenzar con mayúscula
      publish:
        summary: a new pet has been added # No conforme, no comienza con mayúscula
        message:
          contentType: application/json
          name: PetAddedEvent
          payload:
            type: object
            properties:
              id:
                type: string
                description: Unique identifier for the pet
              name:
                type: string
                description: Name of the pet

Compliant Solution (AsyncAPI)

  asyncapi: '2.3.0'
  info:
    version: '1.0.0'
    title: Swagger Petstore Events
  servers:
    production:
      url: broker.mycompany.com
      protocol: mqtt
  channels:
    petAdded:
      description: "An event for when a pet is added" # Ahora comienza con mayúscula
      publish:
        summary: A new pet has been added # Conforme, comienza con mayúscula
        message:
          contentType: application/json
          name: PetAddedEvent
          payload:
            type: object
            properties:
              id:
                type: string
                description: Unique identifier for the pet
              name:
                type: string
                description: Name of the pet
